Pred pričetkom v terminalu, najlažje kar v Visual Studio Code, ki direktno kaže na mapo projekta napišemo: 


```
pip install -r requirements.txt
```

Ko se namestijo vsi paketi pa samo še poženemo ukaz:

```
python app.py
```

    In aplikacija bo tekla na:

[http://127.0.0.1:5000](http://127.0.0.1:5000)

Ko končamo z uporabo pa samo še ugasnemo Flask z ukazom:

`ctrl c`
